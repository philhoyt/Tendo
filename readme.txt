== Description ==

Tendo is a Clean and Minimalist theme built for the Full Site Editing features included in WordPress 5.9. Take advantage of Global Style and Template editing to customize the look and feel of Tendo to meet match your own unique needs. Use block patterns to better display your content with more to come in future iterations. You must have the Gutenberg plugin active to use Tendo before WordPress 5.9 is released.
